45-Minute Unity Mini Game: 2D Clicker Challenge

Spielidee
Das Spiel ist ein extrem simpler 2D-Clicker.
Der Spieler sieht ein einzelnes 2D-Objekt auf dem Bildschirm.
Das Ziel des Spiels ist es, dieses Objekt insgesamt 10-mal anzuklicken.
Bei jedem Klick erhöht sich der Punktestand (Score) um eins.
Sobald 10 Punkte erreicht sind, wird das Spiel angehalten und eine Erfolgsmeldung („You Won!“) eingeblendet.

Steuerung
Klicke auf das 2D-Objekt, um Punkte zu sammeln.
